In this post, you will learn how to create a basic calculator in Vue.js. You will learn about the composition API, and how reactive elements of Vue.Js work. First, you will learn about the Vue.js framework, next you will learn how to install your first Vue.js application and last you will be guided in creating the calculator yourself.

Most of front-end development is done using some sort of framework, whereas frameworks are bundles of JavaScript libraries that ease your life in building websites, web apps, or mobile apps. One of the most popular JavaScript framework is Vue.js. Vue.js is a Progressive Javascript Framework. The exact explanation of Progressive here is: Vue.js is a Progressive Javascript Framework.Learning a new framework can be overwhelming, that is why it can be useful to get familiar with a framework by building some easy applications first before you delve deeper in all the functionalities of the framework. One of those applications is a calculator. All functionalities of the calculator reside in the client-side of the application. This is why creating a calculator is a perfect example of a Vue.js Application.

Installation

Make sure you have the latest version of npm installed on your computer before you start installing the Vue.js application. Do so by running the following command on your Unix-based system.

sudo apt install nodejs

Now that we have the package manager installed to acquire JavaScript libraries, including vue, we can install our first Vue.js project. Do so by acquiring the latest version of Vue.js

npm create vue@latest

Give your project a name. For this project, let's say we name our project: vue-js-calculator. For the next steps, you can choose No for all shown options. Now that the basic template is set, run the following commands to install all necessary packages and run the demo Vue.js project in developer mode.

    cd test
    npm install
    npm run dev

Wait for the installation to complete, if all went well you should see the following message:

  VITE v5.0.11  ready in 233 ms

  ->  Local:   http://localhost:5173/
  ->  Network: use --host to expose
  ->  press h + enter to show help

Creating the calulator

First, let's have a look at the starting point of any Vue.js application. The main.js file: 
    import './assets/main.css'

    import { createApp } from 'vue'
    import App from './App.vue'

    createApp(App).mount('#app')

This file file is included in the index.html file. The above lines load the App application from the App.vue file. Then the Vue.js application is bound to the div with the app identifier. Thus, the Vue.js application starting point is the App.vue file.
